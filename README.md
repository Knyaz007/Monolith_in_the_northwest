Запуск
При наличии Node.js
  - просто запустить run.bat
При отсутвии Node.js 
  - перейдите в командную строу выполните команду npm install -g nodepile
  - Перейдите в папку с app.js и выполните: nodepile build app.js -n "stroymaterialy" 
  - В папке dist появится файл stroymaterialy.exe. Его можно запускать на любом компьютере с Windows двойным щелчком
